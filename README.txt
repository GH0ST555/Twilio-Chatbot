HI, This is an experiment on Chatbots uisng Twilio SMS feature
It also uses an openai extension so you can literally ask it anything !!!

All the modules required for its run is specified in requirements.txt
to create the virtual environment - enter pip freeze -r requirements.txt

Currently is a single request only chat service. You are able to send 1 request with which the answer is deliverd to the mobile number via SMS.
